Using ARGalaxy

Antigen receptor Galaxy (ARGalaxy) is a tool to automate the analysis and visual reporting of T- and B-cell receptor repertoire sequencing data.

ARGalaxy consists of four parts: the demultiplex tool, the IMGT concatenate tool, the immune repertoire pipeline and the SHM&CSR pipeline. Together they allow the analysis of all different aspects of the immune repertoire. All pipelines can be run independently or combined, dependent on the available data and the question of interest

Demultiplex tool

The demultiplex tool demultiplexes .sff files with the option of sequence trimming with both downloadable FASTA and FASTQ files available as output. This tool is based on the freely available sff2fastq (https://github.com/indraniel/sff2fastq), and FASTX-Toolkit software (http://hannonlab.cshl.edu­ /fastxtoolkit/commandline.html), and includes the FastQC tool to check the quality of your data (http://www.bioinformatics.­babraham.ac.uk/projects/fastqc/).

IMGT concatenate tool

The IMGT concatenate tool can be used to combine multiple .txz files (as downloaded from IMGT/High V-Quest http://www.imgt.org/HighV-QUEST ) into a single .txz file.

Immune repertoire pipeline

The immune repertoire pipeline is an extended version of our previously published IGGalaxy virtual machine (Moorhouse et al, 2014) which was developed to visualize V(D)J gene usage. It now allows analysis of not only B-cell, but also T-cell receptor rearrangements and allows the visualization of CDR3 characteristics (length and amino acid usage), junction characteristics and the calculation of the diversity of the immune repertoire (Boyd et al, 2009). The Immune Repertoire pipeline requires fasta files, IMGT/HighV-QUEST output files, or output files from the SHM&CSR pipeline as input files and gives results as a set of  tables and graphs accessible in html format.

SHM&CSR pipeline

The SHM&CSR pipeline can be used for the analysis of somatic hypermutations (SHM) and/or class switch recombination (CSR) in B-cell receptor rearrangements. This SHM&CSR pipeline provides the analysis of SHM levels and patterns, antigen selection (including BASELINe (Yaari et al, 2012)), clonality (using Change-O (Gupta et al, 2015)) and CSR. It requires IMGT/HighV-QUEST .txz files as  input files, parses the relevant information from these files, and provides a html output as a set of tables and graphs.

Test data sets

The test data set can be imported from the 'Data libraries' under 'Shared data' or by using this link: https://bioinf-galaxian.erasmusmc.nl/argalaxy/library/list
Histories in which the test data sets are used in the demultiplex tool, the IMGT concatenate tool, the immune repertoire pipeline or the SHM&CSR pipeline can be imported from the 'Histories' under 'Shared data' or by using this link: https://bioinf-galaxian.erasmusmc.nl/argalaxy/u/davidvanzessen/h/test-data-sets
